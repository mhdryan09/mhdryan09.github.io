About Me

Muhammad Ryan Pranata | Front End Web Developer

Perkenalkan nama saya Muhammad Ryan Pranata. Saat ini, saya sedang bekerja di salah satu Start-Up di Yogyakarta. Pada sela-sela luang waktu setelah bekerja, saya menekuni bidang yang saya sukai saat ini yaitu pemrograman, terkhusus pada teknologi di bagian Front End Web.

Saat ini, saya sedang belajar mengenai Javascript ES6 dan terkadang mecoba sesekali melakukan slicing design to code (menjadikan design ke dalam bentuk halaman website statis). Walaupun diluar sana sedang ramai menggunakan Framework Javascript seperti React/Vue/Angular, saya tidak terlalu ingin buru-buru masuk kesana. Hal yang saya perkuat saat ini adalah fundamental pemrograman javascript itu sendiri. Agar jika suatu saat ingin berpindah-pindah framework, tidak butuh waktu lama. Disamping itu semua, saya antusias dan merasa selalu ingin belajar apabila ada teknologi baru.
